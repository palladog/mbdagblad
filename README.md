## Requirements
* Python 3
* Pip
* Virtual Environment

## Installation
1. In the console, enter the main directory
2. Create a virtual environment, e.g. using Python Virtual Environment
```
$ python3 -m venv venv
$ source venv/bin/activate
```
3. Install dependencies, e.g. using pip:
```
(venv) $ pip install -r requirements.txt
```
4. Run the Python file `app.py`, either in debug mode or straight-forward:
```
(venv) $ python3 app.py
```
